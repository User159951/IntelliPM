# IntelliPM Frontend

## Project info

This is the React + Vite frontend for **IntelliPM**, an intelligent project management dashboard with AI-powered agents.

## How can I edit this code?

You can work with any IDE you prefer. The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How can I deploy this project?

You can deploy the built `dist` folder to any static hosting provider (Azure Static Web Apps, Vercel, Netlify, GitHub Pages, etc.).
